Human Resources Analytics Dashboard
Overview

The Human Resources Analytics Dashboard is an interactive Power BI project developed as part of my internship. The dashboard helps HR professionals monitor workforce performance, recruitment efficiency, employee retention, and hiring trends. It provides data-driven insights to support strategic HR decision-making.

Objective

The objective of this project is to analyze and optimize HR processes by visualizing key workforce metrics and enabling informed business decisions through interactive dashboards.

Key Features
Recruitment Metrics Analysis
Track total applicants, hires, and recruitment performance.
Monitor recruitment efficiency and hiring trends.

Employee Turnover Analysis
Analyze employee attrition rates.
Identify departments and job roles with higher turnover.

Employee Satisfaction & Performance
Evaluate employee satisfaction scores.
Compare performance ratings across departments and job roles.
Identify patterns affecting employee retention.

Predictive Analytics
Forecast future hiring requirements using historical workforce data.
Support HR planning and resource allocation.

Dashboard Insights
Recruitment trends over time.
Employee attrition by department, gender, and job role.
Employee satisfaction and performance comparison.
Workforce demographics and distribution.
Hiring forecast for future workforce planning.

Tools & Technologies
Power BI
Microsoft Excel (Data Source)
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Data Visualization

Outcomes
Improved visibility into recruitment and employee turnover.
Identified workforce trends affecting employee retention.
Enabled HR managers to make data-driven decisions.
Supported workforce planning through predictive hiring analysis.
